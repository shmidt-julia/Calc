# Calc

Інструкція по встановленню, компіляції та запуску програми:

1. Встановити Java SE JDK 8 Development Kit (Рекомендовано! Можливо встановлення більш нової версії)
2. Розпакувати архів з програмою на локальний диск (Наприклад, D:)
3. Запустити консоль та перейти в директорію(категорію) обраного диска (D:\>cd Calc-master\src)
4. Виконати компіляцію файла програми "EverywhereCalculator.java" (D:\Calc-master\src>javac EverywhereCalculator.java)
5. Запустити програму "EverywhereCalculator.java" (D:\Calc-master\src>java EverywhereCalculator)


----------------------------------------------------------------------------------------------------------------------

Інструкція щодо основних правил використання програми:

1. Для обчислення необхідно ввести три аргументи ЧЕРЕЗ ПРОБІЛ у форматі (operant1 action operant2, like 2 + 2 )
2. Допустимі для введення цілі числа більші за нуль
3. Для обчислення доступно п'ять математичних операцій (+, -, *, /, %)
4. Результат обчислення відображається на екрані у форматі (RESULT IS: 4) і одразу доступне нове обчислення
5. Щоб вийти з програми необхідно використати будь-яку з двох команди ("q" або "exit")
